# 中文 AI 写作指纹模式库

用于步骤 4b 的 AI 去痕审查。扫描以下模式，命中则标记并建议替换。

## 1. 过度强调意义

| 模式 | 示例 | 建议 |
|------|------|------|
| "彰显了" | "这一变化彰显了..." | "这一变化表明..." |
| "具有里程碑意义" | "具有里程碑意义的突破" | 删除或给出具体数据 |
| "深远的影响" | "产生了深远的影响" | "影响了..." |
| "不可估量的" | "不可估量的损失" | 给出具体数据 |
| "划时代的" | "划时代的变革" | 删除修饰词 |
| "里程碑" | "这是一个里程碑" | 删除 |

## 2. 空泛引导

| 模式 | 示例 | 建议 |
|------|------|------|
| "引发了广泛关注" | "这一事件引发了广泛关注" | 删除或引用具体来源 |
| "引起了热议" | "在社交媒体上引起了热议" | 引用具体讨论和观点 |
| "引发了人们的思考" | "这引发了人们的思考" | 给出具体思考方向 |
| "备受关注" | "备受关注的..." | 删除或给出具体数据 |
| "持续升温" | "讨论持续升温" | 给出具体数据或引用 |
| "成为了焦点" | "成为了舆论焦点" | 删除 |

## 3. 机械排比

| 模式 | 示例 | 建议 |
|------|------|------|
| "不仅是...更是...也是..." | "这不仅是...更是...也是..." | 改为简洁单句 |
| "一方面...另一方面...此外..." | 三句以上排比 | 精简到两个核心点 |
| "从...到...从...到..." | "从个人到组织，从技术到文化" | 删除套话 |
| "无论...还是...都..." | "无论是企业还是个人，都..." | 删除 |
| "与其说...不如说..." | "与其说X，不如说Y" | 直接陈述观点 |

## 4. 堆砌修饰

| 模式 | 示例 | 建议 |
|------|------|------|
| "作为一个...的...的..." | "作为一个有着深厚技术背景的..." | 拆分或精简 |
| "在...的大背景下" | "在数字化转型的大背景下" | 删除 |
| "纵观...的历史" | "纵观互联网发展的历史" | 直接进入具体案例 |
| "毋庸置疑" | "毋庸置疑的是..." | 删除 |
| "不言而喻" | "不言而喻" | 删除 |
| "众所周知" | "众所周知" | 删除，直接陈述 |

## 5. 空泛结论

| 模式 | 示例 | 建议 |
|------|------|------|
| "值得我们深思" | "这值得我们深思" | 给出具体行动建议 |
| "任重而道远" | "AI 的发展任重而道远" | 给出具体方向 |
| "未来可期" | "未来可期" | 删除 |
| "拭目以待" | "让我们拭目以待" | 删除 |
| "还需要进一步探索" | "还需要进一步探索" | 给出具体探索方向 |
| "没有标准答案" | "这个问题没有标准答案" | 给出你的判断 |

## 6. 过度谦虚/免责

| 模式 | 示例 | 建议 |
|------|------|------|
| "本文仅代表个人观点" | "本文仅代表个人观点" | 删除 |
| "如有不当之处请指正" | "如有不当之处请指正" | 删除 |
| "可能我理解有误" | "可能我理解有误" | 直接表达观点 |
| "抛砖引玉" | "仅以此文抛砖引玉" | 删除 |
| "粗浅的看法" | "以上是我粗浅的看法" | 删除 |

## 7. 模板化开头

| 模式 | 示例 | 建议 |
|------|------|------|
| "随着...的发展" | "随着 AI 技术的快速发展" | 用具体场景/案例开场 |
| "在当今...的时代" | "在当今信息爆炸的时代" | 删除 |
| "近年来" | "近年来，AI 取得了巨大进步" | 直接进入具体事件 |
| "众所周知" | "众所周知，AI 正在改变一切" | 删除 |
| "你有没有想过" | "你有没有想过..." | 改为具体场景描述 |
| "在这个...的时代" | "在这个瞬息万变的时代" | 删除 |

## 8. 其他常见模式

| 模式 | 示例 | 建议 |
|------|------|------|
| "让我们回到最初的问题" | "让我们回到最初的问题" | 直接陈述 |
| "不难发现" | "不难发现，..." | 删除，直接陈述 |
| "由此可见" | "由此可见，..." | 删除 |
| "综上所述" | "综上所述，..." | 删除 |
| "总而言之" | "总而言之，..." | 删除 |
| "值得注意的是" | "值得注意的是，..." | 删除，直接陈述 |
| "事实上" | "事实上，..." | 删除或保留（如果确实有转折） |
| "换句话说" | "换句话说，..." | 删除，直接说清楚 |

## 使用指南

审查时逐条扫描上述模式，命中后：
1. 列出命中位置和原文
2. 给出建议替换
3. 用户确认后修改

不要机械替换——有些情况下这些表达是合理的（如引用他人原话时）。
