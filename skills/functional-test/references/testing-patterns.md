# Testing Patterns

各类型应用的结构化测试模式参考，用于指导 Phase 1 应用分析和 Phase 2 测试计划生成。

## Web 应用

### 导航测试
- **页面可达性**: 每个路由/页面至少 1 条 Golden Path 用例
- **刷新保持**: 刷新页面后停留在当前页面、保持表单输入
- **后退/前进**: 浏览器后退/前进按钮状态正确恢复
- **深链接**: 直接访问子页面 URL 正常加载（不依赖前置导航）

### 表单测试
- **正常提交**: 填写所有必填字段，提交成功
- **空提交**: 不填任何字段，显示字段验证错误
- **部分必填**: 只填部分字段，提示缺失必填项
- **特殊字符**: 输入 `<script>alert(1)</script>`、`' OR 1=1 --`，正确转义
- **超长输入**: 输入 10000 字符，截断或正确显示，不崩溃
- **文件上传**: 支持/不支持的文件类型、超大文件

### API 交互测试
- **正常调用**: API 返回 200，数据正确渲染
- **网络断开**: DevTools 离线模式操作，显示错误提示
- **超时**: 模拟慢网络（Slow 3G），显示加载状态，超时后提示
- **错误状态**: API 返回 400/401/403/500，对应错误处理正确

### 数据列表测试
- **空数据**: 显示空状态提示，非白屏
- **单条数据**: 正确渲染
- **大数据量**: 1000+ 条记录，有分页或虚拟滚动，不卡顿
- **搜索/过滤**: 输入搜索词，列表正确过滤
- **排序**: 点击列头排序，升序/降序正确

## Electron 桌面应用

### 启动测试
- **正常启动**: 主窗口正常显示，无控制台错误
- **开机自启**: 设置开机自启后重启，应用自动启动
- **多实例**: 已运行时再次打开，聚焦已有窗口而非启动新实例

### IPC 通信测试
- **正常调用**: renderer → main 请求发送，响应正确接收
- **主进程失败**: 主进程处理失败，渲染进程收到错误，不崩溃
- **超时**: IPC 调用超时，渲染进程有超时处理

### 数据持久化测试
- **设置保留**: 修改设置后重启应用，设置保留
- **历史数据**: 加载大量历史数据，不卡顿
- **异常恢复**: 数据文件损坏时，应用不崩溃，提示修复

### 窗口/托盘测试
- **托盘图标**: 启动后系统托盘图标正常显示
- **托盘菜单**: 右键托盘菜单项正确显示和响应
- **窗口调整**: 调整窗口大小，内容自适应，不溢出
- **最小化/恢复**: 状态正确恢复

## CLI 工具

### 启动测试
- **--help**: 显示帮助信息，退出码 0
- **--version**: 显示版本号，退出码 0
- **无参数**: 显示简要用法或进入交互模式

### 子命令测试
- **正常执行**: 子命令正常输出，退出码 0
- **子命令 --help**: 显示子命令帮助，退出码 0

### 参数测试
- **必填参数缺失**: 错误提示，退出码 != 0
- **无效值**: 输入无效类型/范围外的值，错误提示，退出码 != 0
- **参数组合**: 互斥参数同时使用，错误提示

### 输出/管道测试
- **正常输出**: 输出格式正确，人类可读
- **JSON 输出**: `--json` 输出合法 JSON
- **管道输入**: `echo "data" | cmd` 正确读取 stdin
- **管道输出**: `cmd | grep pattern` 输出可被下游工具消费

### 错误处理测试
- **权限不足**: 无权限时执行，权限错误提示，退出码 1
- **文件不存在**: 指定不存在的文件，友好错误提示
- **Ctrl+C**: 运行中 Ctrl+C，优雅退出，退出码 130

## 专项测试模式

### Accessibility
- 所有图片有 alt 文本
- tab 顺序逻辑清晰
- 焦点可见（outline 不缺失）
- aria-label 用于无文本按钮
- 对比度符合 WCAG AA（4.5:1）

### Performance
- 首次内容绘制 < 1.5s
- 页面可交互 < 2s
- API 响应 < 500ms
- 列表 1000 项渲染 < 1s
- 内存无持续增长（长时间操作）

### Security
- 用户输入不直接渲染到 innerHTML
- API 请求使用 HTTPS
- Token 不在 URL 中传递
- 敏感操作需要确认（删除、支付）
