# Bug Report Template

## Bug 报告格式

```markdown
## BUG-{编号} — {简短标题}

| 字段 | 值 |
|------|-----|
| 严重度 | BLOCKER / CRITICAL / HIGH / MEDIUM / LOW |
| 模块 | 所属功能模块 |
| 关联用例 | T-XXX |
| 环境 | 系统、浏览器/应用版本、Node 版本等 |
| 发现日期 | YYYY-MM-DD |

### 复现步骤
1. 打开 ...
2. 点击 ...
3. 输入 ...
4. 观察 ...

### 预期行为
{描述预期结果}

### 实际行为
{描述实际结果}

### 复现率
{必现 / 偶现(X%) / 仅首次}

### 截图/日志
{错误截图、控制台日志、网络请求等}

### 根因分析（可选）
{如果已定位到代码层面的原因}

### 修复建议（可选）
{建议的修复方案}
```

## 严重度定义

| 严重度 | 定义 | 修复时限 |
|--------|------|----------|
| BLOCKER | 应用完全不可用，阻塞所有测试 | 立即修复 |
| CRITICAL | 核心功能完全不可用 | 本次迭代 |
| HIGH | 重要功能异常但可绕过 | 下次迭代 |
| MEDIUM | 非核心功能异常 | 规划排期 |
| LOW | 体验优化建议 | backlog |

## 常见 Bug 分类

### 功能性 Bug
- 功能不生效（按钮点击无反应、表单提交失败）
- 功能行为错误（排序不正确、过滤无效）
- 数据错误（显示错误数据、计算结果不对）

### UI Bug
- 样式错乱（元素重叠、间距错误、颜色错误）
- 响应式问题（移动端布局错误、窗口调整不响应）
- 主题问题（暗色模式未适配、主题切换闪烁）

### 性能 Bug
- 页面加载过慢（>3s）
- 操作响应延迟（>1s）
- 内存泄漏（长时间使用后变卡）
- CPU 占用过高（风扇狂转）

### 兼容 Bug
- 浏览器不兼容（仅 Chrome 正常，Firefox/Safari 异常）
- 系统版本不兼容（仅 macOS 正常，Windows 异常）
- 分辨率不兼容（特定分辨率下显示错误）
